Wireshark Packet Capture & Analysis Lab
Overview
-------------------

This project demonstrates basic network analysis using Wireshark, including TCP handshake inspection, DNS queries, and HTTP traffic analysis.

The official website of Wireshark is https://www.wireshark.org

Installation instructions can be found in the "Install" file and in the
Developer's Guide at https://www.wireshark.org/docs/wsdg_html_chunked/

Tools Used
-------------------
  - Wireshark
  - Windows 11 (endpoint)

Objectives
-------------------
  - Capture real network traffic
  - Filter and analyze TCP, DNS, and HTTP packets
  - Identify TCP 3-way handshake
  - Decode DNS queries & responses
  - Inspect HTTP GET/response behavior
